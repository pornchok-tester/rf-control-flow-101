Control Flow ใน Robot Framework

✅ Tested: RF 7.3.2 ⚠️ ต้องใช้ RF 5.0+

เอกสารนี้พา 4 เรื่องที่ทำให้ test ไม่ซ้ำซ้อน: ตัดสินใจตามเงื่อนไข วนซ้ำข้อมูล รอจนกว่าจะสำเร็จ และเอาไปใช้กับ Selenium จริง

Prerequisites

1. โครงสร้างพื้นฐานของ RF

ไฟล์ .robot แบ่งเป็น section ด้วย *** ... ***:

*** Settings ***
Library    Collections          # import library เพิ่มเติม

*** Variables ***
${NAME}    Alice                # ตัวแปร global

*** Keywords ***
Say Hello
    [Arguments]    ${name}      # keyword รับ argument
    Log    Hello, ${name}!      # สั่งให้ทำอะไรบางอย่าง

*** Test Cases ***
My First Test
    Say Hello    Bob            # เรียกใช้ keyword
  • Test Case คือหน่วยทดสอบ 1 ชุด — RF รันทีละ Test Case
  • Keyword คือ "คำสั่งที่นิยามเอง" เหมือน function — เรียกซ้ำได้
  • *** Settings *** ใช้ import Library หรือตั้งค่า suite

2. ตัวแปร ${variable} และ @{list}

# ${variable} — ตัวแปรทั่วไป
${score}=    Set Variable    85
${name}=     Set Variable    Alice
${flag}=     Set Variable    ${True}

# @{list} — ตัวแปรแบบ list
@{fruits}=    Create List    apple    banana    cherry
Log    ${fruits}[0]    # → apple
⚠️ จุดสำคัญ

@{fruits} ใน FOR loop จะ unpack รายการออกมาทีละชิ้น แต่ ${fruits} จะส่ง list ทั้งก้อนเป็น item เดียว — เรื่องนี้จะเจออีกในบทที่ 2

3. Built-in Keywords ที่ใช้บ่อยในเอกสารนี้

Keywordทำอะไรตัวอย่าง
Logprint ข้อความลง logLog Hello World
Set Variableกำหนดค่าให้ตัวแปร${x}= Set Variable 10
Evaluateคำนวณ Python expression${sum}= Evaluate ${a} + ${b}
Should Be Equalassert ว่าค่าสองอันเท่ากันShould Be Equal ${x} ${y}
Create Listสร้าง list ใหม่${items}= Create List a b c
Get Lengthนับจำนวน item ใน list${len}= Get Length ${items}
Run Keyword And Return Statusรัน keyword แล้วคืน True/False${ok}= Run Keyword And Return Status ...

หมายเหตุ: Evaluate สำคัญมากใน WHILE loop เพราะใช้คำนวณค่าใหม่ทุกรอบ เช่น ${counter}= Evaluate ${counter} + 1

กฎสำคัญก่อนเริ่ม

#กฎตัวอย่าง
1 ทุก keyword ต้องเป็น UPPERCASE IF ELSE IF ELSE FOR WHILE BREAK CONTINUE END
2 ทุก block ต้องปิดด้วย END IF ... END, FOR ... END, WHILE ... END
3 Indent 4 spaces ภายใน block keyword ข้างใน IF/FOR/WHILE ต้องเว้น 4 spaces
4 Condition ใช้ Python expression IF ${x} > 10 and ${y} < 5
